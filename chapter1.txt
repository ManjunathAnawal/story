Local repository
--------------------------------------------------------------------

Git init   			// working directory
ls -a     			// check hidden files
Git status 			// check untracked ,changed containt,commits of files 
Git add file_name 		// add file to staging area 
Git add .        		 // add all files to staging area
Git commit -m "commit msg"	 // commit tracked file/files  to local repo
Git log  			// check previous commits
Git diff file_name 		// difference between file
